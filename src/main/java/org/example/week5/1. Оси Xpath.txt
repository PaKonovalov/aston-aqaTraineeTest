child:: — содержит множество элементов-потомков (элементов, расположенных на один уровень ниже). Это название
сокращается полностью, то есть его можно вовсе опускать.

parent:: — содержит элемент-преход на один уровень назад. Это обращение можно заменить на ..

following-sibling:: — содержит множество братских элементов того же уровня, следующих за текущим слоем.

preceding-sibling:: — содержит множество братских элементов того же уровня, предшествующих текущему слою
